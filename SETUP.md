# ocr-tool — Bootstrap for Claude Code

Скопируй и вставь **весь текст ниже** в Claude Code из папки этого проекта.
Claude сам выполнит установку и настройку.

---

```
Я только что склонировал ocr-tool. Сделай следующее:

1. Прочитай CLAUDE.md — там все инструкции.
2. Проверь, установлен ли пакет, и если нет — установи.
3. После установки MCP-сервер из .mcp.json заработает автоматически.
4. Сделай тестовый OCR: создай картинку с текстом и распознай её.
5. Расскажи что получилось и какие инструменты теперь доступны.
```

После этого `ocr_image` и `ocr_screen` будут доступны как нативные инструменты.

## Ручная установка

```bash
git clone <repo-url> && cd ocr-tool
pip install -e ".[all]"
```

Claude Code сам подхватит `.mcp.json` в папке проекта.
